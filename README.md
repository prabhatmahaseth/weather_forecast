# Weather Forecast Webpage

A modern, responsive weather forecast demo built using **HTML5**, **CSS3 (Tailwind CSS + custom animations)**, and **JavaScript**. This project features a clean UI with animated weather icons, a 3-day forecast section, interactive city search, dark mode toggle, and a modal for detailed information. All weather data is mocked for demonstration purposes.

## Features

- Current weather display with city name, temperature, condition, humidity, and wind speed
- Animated weather icons (sun rotation, moving clouds, rain drops)
- 3-day forecast cards showing date, icon, min/max temperature, and description
- Dark mode toggle
- Dynamic background gradients based on weather condition
- Search bar with simulated data refresh and a refresh animation
- Modal popup for additional details when forecast cards are clicked
- Responsive layout using Flexbox and Tailwind CSS
- Smooth transitions and hover animations

## Files

- `index.html` &ndash; main HTML file containing the layout and links to styles/scripts
- `style.css` &ndash; custom styles, animations, and dark mode support
- `script.js` &ndash; JavaScript for mock data, UI updates, and interaction logic
- `README.md` &ndash; this documentation

## Usage

Open `index.html` in a browser. Use the search box to enter any city name and click **Search** or **Refresh** for new mocked weather data. Toggle dark mode with the moon switch and click on forecast cards for more information.
